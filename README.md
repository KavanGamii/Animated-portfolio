# Animated-portfolio

Cinematic personal portfolio for Kavan Gami — a UI Developer & Web Designer.

Built with **TanStack Start** (React 19 + SSR), **Vite**, **Tailwind CSS v4**, and motion powered by **Framer Motion**, **GSAP / ScrollTrigger**, and **Lenis** smooth scroll.

## Development

```bash
bun install
bun dev
```

## Build

```bash
bun run build
bun run preview
```
